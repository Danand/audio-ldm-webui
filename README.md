# AudioLDM 2: Web UI

Simple web UI for AudioLDM 2, built with [Streamlit](https://streamlit.io/).

## Prerequisites

- [Python](https://www.python.org/downloads)

## How to install

```bash
git clone git@github.com:Danand/audio-ldm-webui.git
cd audio-ldm-webui
chmod +x webui.sh
```

## How to launch

```bash
./webui.sh
```
